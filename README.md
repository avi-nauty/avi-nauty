<h2 align="center">🚀 Projects</h2>

### 🧠 [Brain Tumor Detection from MRI Images](https://github.com/avi-nauty/Medical-Image-Classification-using-Machine-Learning)
`Python` `TensorFlow` `Keras` `Scikit-learn` `Kaggle`

End-to-end ML pipeline for binary MRI classification (Tumor vs. Healthy). Fused deep features from ResNet50, VGG16, and EfficientNetB0 into a 3,840-dimensional vector, applied Black Hole Optimization (BHO) for feature selection (~2,000 optimal features), and trained a custom Opt-SGD Logistic Regression classifier from scratch. Achieved **96.9% accuracy** across three MRI datasets. Paper submitted to **IEEE Intelligent Systems**.

### 🚦 [Traffic Sign Recognition Using CNN](https://github.com/avi-nauty/Traffic-Sign-Recognition-Using-CNN-Architecture)
`Python` `TensorFlow` `OpenCV`

CNN-based multi-class image classification system recognizing 43 traffic sign categories from the GTSRB dataset (50,000+ images). Implemented preprocessing pipelines, data augmentation, regularization, and early stopping. Achieved ~**97% test accuracy** within 20–30 epochs.
